# Automate Trade EA – Documentation

## Logic

Expert Advisor for automating trades on MetaTrader 5, built with the YM (Execute/Position) library. Logic and entry/exit rules are implemented in the source.

## Parameters

See the input parameters in the EA source (`Automate trade.mq5`) and in the MetaTrader inputs window when attaching to a chart.

## Features

- Trade automation via YM library
- Configurable inputs

## Risks and limitations

- No guarantee of profitability. Backtest and use at your own risk.
- Depends on external YM library; ensure Include path is set correctly.
- For educational use only unless otherwise agreed.
