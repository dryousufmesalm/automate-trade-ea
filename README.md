# Automate Trade EA

MetaTrader 5 Expert Advisor for trade automation using the YM library.

**Author:** Yousuf Mesalm | www.yousufmesalm.com | WhatsApp +201006179048 | Upwork: https://www.upwork.com/freelancers/youssefmesalm

## Requirements

- MetaTrader 5
- YM library in `MQL5/Include/YM` (terminal Include path must point to your terminal’s MQL5 folder)

## Installation

1. Copy the EA `.mq5` file into `MQL5/Experts/` (or a subfolder).
2. Ensure the terminal’s **Include** path includes your `MQL5` directory so that `Include/YM` is found.

## Compilation

Open the file in MetaEditor and compile (F7). Include path must be the terminal’s MQL5 root (e.g. `C:\...\MQL5`).

## Usage

Attach the EA to a chart. Configure inputs as needed (see DOCUMENTATION.md).
