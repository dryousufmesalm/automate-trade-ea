# Automate Trade EA

**Automate your MetaTrader 5 trading with a flexible, YM-based Expert Advisor.**

This EA uses the YM (Execute/Position) library for reliable order execution and position management on MT5. Attach it to a chart, set your inputs, and let it run 24/5.

---

## Why use this EA?

- **Fully automated** – Runs continuously once attached.
- **YM-based execution** – Robust order and position handling.
- **Customizable** – Adapt inputs and logic to your strategy.
- **MT5 native** – Built for MetaTrader 5.

---

## Who is it for?

Traders and developers who want automated execution with the YM library, or a clean template for custom EAs.

---

## Requirements

- MetaTrader 5
- YM library in `MQL5/Include/YM` (set your terminal Include path to the MQL5 folder that contains it).

---

## Installation

1. Copy the `.mq5` file into `MQL5/Experts/` (or a subfolder).
2. In MetaEditor, add your terminal MQL5 root to the Include path so `Include/YM` is found.
3. Open the EA in MetaEditor and compile (F7).

---

## Quick start

1. In MT5, drag the EA from the Navigator onto your chart.
2. Enable **Allow Algo Trading** and set lot size, magic number, and other inputs.
3. Click OK to start.
